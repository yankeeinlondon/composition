# LSP Server Implementation Videos

These tutorials focus on implementing Language Server Protocol (LSP) servers in various languages.

## Rust LSP Server Videos

These tutorials focus on implementing a Language Server Protocol (LSP) server using Rust.

- **Build your own LSP using RUST | Language Server Protocol**
  - **Description:** A step-by-step tutorial on implementing a basic LSP from scratch using Rust and JSON-RPC.
  - **Link:** [https://www.youtube.com/watch?v=dRxbqca6p60](https://www.youtube.com/watch?v=dRxbqca6p60)
  - **Timeline Highlights:** Dependencies (03:58), Server implementation (05:40), Client implementation (18:32), Run and Test (27:25).

- **Let's read the Rust LSP server source code**
  - **Description:** This video dives into the source code of a Rust LSP server implementation to understand its inner workings.
  - **Link:** [https://www.youtube.com/watch?v=PNCldjGJTcU](https://www.youtube.com/watch?v=PNCldjGJTcU)

- **Writing A Language Server Part 2** (TheVimeagen)
  - **Description:** Part 2 of a series exploring the process of writing a language server, generally focusing on Rust.
  - **Link:** [https://www.youtube.com/watch?v=JC4vmHVt59s](https://www.youtube.com/watch?v=JC4vmHVt59s)
